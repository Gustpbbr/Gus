## Convenções desta área

- **Link salvo:** `capturado/links/[titulo-curto].md` com resumo do conteúdo + URL original em `fonte:`.
- **Ideia solta:** `capturado/ideias/[tema].md`. Se a ideia virar recorrente, considerar promover pra projeto próprio.
- **Misc:** `capturado/misc/[descricao-curta].md` — último recurso quando não encaixa em nada.
- **Wikilinks:** quando a captura tem relação com projeto ativo, linkar (`[[phronesis-bench]]`, `[[ter]]`, etc.) pra não ficar órfã.

## Fluxo quando chegar captura

1. Bot decide se já existe categoria melhor em outra área. Se sim, salvar lá (não em `capturado/`).
2. Se é realmente solto, escolher `links/`, `ideias/` ou `misc/`.
3. Adicionar wikilinks pra conexões óbvias.
4. Atualizar este índice.
5. Periodicamente (mensal, via revisão) — promover capturas recorrentes pra áreas próprias.
