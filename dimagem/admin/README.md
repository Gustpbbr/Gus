# dimagem/admin/

Pendências administrativas e operacionais do Dimagem — burocracia,
contatos, convênios, equipamentos, agenda institucional.

## Estrutura sugerida

- **`pendencias.md`** — lista viva de pendências administrativas
- **`contatos.md`** — fornecedores, técnicos, convênios (telefones,
  emails) — **CUIDADO LGPD**: dados de contato de terceiros podem
  precisar ir em `sensivel/` em vez daqui
- **`equipamentos.md`** — inventário de equipamentos, manutenções
  programadas
- **`convenios-relacionamento.md`** — status do relacionamento com
  cada convênio (auditorias, pendências, contatos)

## Diferença das outras subpastas

- **`admin/`**: gestão da clínica, não-clínico
- **`dia/`**: operacional diário (lista de pacientes do dia)
- **`casos/`**: didático (pseudônimo)
- **`protocolos/`**: regras técnicas
