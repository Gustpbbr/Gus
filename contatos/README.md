# contatos/

Mapa de relacionamentos do Gustavo.

## ⚠️ ATENÇÃO LGPD

Dados de **terceiros** (telefone, email, endereço) são dados pessoais
deles. Tratamento deve respeitar:

- **Apenas o necessário**: registrar contexto de relacionamento, NÃO
  dossiê.
- **Sensíveis vão pra `sensivel/`**: telefones, endereços, documentos
  de terceiros não vão pro Drive sync.
- **Dado público é OK**: profissão, vínculo (médico endocrino dele,
  professor da disciplina X) podem ficar aqui.

## Estrutura

- **`mapa.md`** — visão consolidada das relações (familiares próximos,
  rede profissional, contatos relevantes pra cada projeto).
- **`<nome-curto>.md`** — opcional, página dedicada pra contato com
  histórico de interações relevantes (não usar como CRM, usar pra
  contatos com contexto recorrente).

## O que registrar aqui (não-sensível)

- Nome e relação ("Cleir, amigo de longa data")
- Vínculo profissional ("endocrinologista que acompanha desde 2024")
- Tópicos de conversa recorrentes
- Datas relevantes (aniversário público se apropriado)

## O que NÃO registrar aqui (vai pra `sensivel/`)

- Telefone, WhatsApp
- Email pessoal
- Endereço residencial
- CPF, RG
- Dados financeiros de terceiros
