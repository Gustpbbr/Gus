- Evidência:
  ```python
  async def gerar_resposta(messages, memory_context=""):
      ...
      system_prompt = _load_system_prompt()
      agora = datetime.now(BRT)            # ← fresh a cada chamada
      dia_semana = DIAS_SEMANA[agora.weekday()]
      system_prompt += f"\n\n## Data e hora atuais\nHoje é {dia_semana}, {agora.strftime('%d/%m/%Y')}, {agora.strftime('%H:%M')}..."
  ```
- O `_load_system_prompt()` tem `@functools.lru_cache` mas só cacheia a **leitura do arquivo** `system_prompt.md`, não o timestamp. A string final (prompt + timestamp) é reconstruída toda vez.

**Não feito:**

**Hipóteses pra data/horário errado (se acontecer de novo):**
